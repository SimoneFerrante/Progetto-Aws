AWS

AWS(Amazon Web Services) e' una piattaforma di cloud computing fornita da Amazon,

include una vasta gamma di servizi cloud come l'elaborazione dei dati, l'archiviazione e la gestione dei database

Inoltre AWS fornisce un'infrastruttura altamente scalabile, sicura e affidabile che consente ai clienti di creare e gestire facilmente le applicazioni,

sfruttando le tecnologie più avanzate del cloud computing senza doversi preoccupare di gestire fisicamente l'hardware.



DOCKER

Docker e' una piattaforma open-source che consente di creare, gestire e distribuire applicazioni in container.

I container sono degli ambienti isolati che contengono tutte le risorse necessarie all'esecuzione dell'applicazione,

tra cui librerie, dipendenze, file di configurazione e il codice stesso dell'applicazione.



HTML

HTML(HyperText Markup Language) e' il linguaggio di markup standard utilizzato per la creazione di pagine web.

E' costituito da una serie di tag che descrivono la struttura e il contenuto della pagina web, come paragrafi, immagini, link e tabelle.

HTML e' un linguaggio di markup semantico, il che significa che ogni tag ha un significato specifico e aiuta i motori di ricerca e dispositivi di assistenza

a comprendere meglio il contenuto della pagina. HTML e' un linguaggio di marcatura molto importante per lo sviluppo web, che forma il fondamento di ogni sito web.



PHP

PHP e' un linguaggio di scripting interpretato utilizzato principalmente per lo sviluppo di applicazioni web lato server.

PHP e' stato progettato per interagire con HTML e con database, come ad esempio MySQL.

Il codice PHP viene eseguito lato server e il risultato e' poi inviato al browser dell'utente.

Tra le sue caratteristiche principali vi sono la facilita' di utilizzo, l'ampia disponibilita' di librerie e framework, la scalabilita' e la sicurezza.

Inoltre, PHP supporta diversi paradigmi di programmazione, tra cui l'orientamento agli oggetti e la programmazione funzionale.



CSS

CSS (Cascading Style Sheets) e' un linguaggio di stile utilizzato per definire l'aspetto estetico di un sito web.

Permette di separare la struttura del contenuto (HTML) dall'aspetto visivo del sito, rendendolo più facilmente gestibile e modificabile.

Con CSS, e' possibile definire i colori, la dimensione e il tipo di carattere dei testi, lo sfondo, il layout e molti altri aspetti del design di un sito.
